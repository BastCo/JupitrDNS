
# JupitrDNS

JupitrDNS is a free security focused DNS service that blocks malware.

- No Logs
- DNSSEC Support
- No EDNS Client Subnet
- High Availability Infrastructure
- Servers Located in Montreal, Canada
- [Network Status](https://status.jupitrdns.com)
