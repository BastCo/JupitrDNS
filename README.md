
# Jupitr DNS

Jupitr DNS is a free hobby DNS service that blocks ads, trackers and malware.

- No logs
- DNSSEC support
- No EDNS Client Subnet
- Servers in North America and Europe
- Network status [stats.jupitrdns.com](https://stats.jupitrdns.com)
